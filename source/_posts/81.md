| 青椒 | 200克| 辅料 |
| 葱 | 适量| 配料 |

| 美极鲜 | 一勺| 配料 |
| 糖 | 半勺| 配料 |
| 五香粉 | 少许| 配料 |
| 盐 | 一小勺| 配料 |
