## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 马铃薯 | 120克| 主料 |
| 面粉 | 120克| 主料 |
| 面粉 | 适量| 辅料 |
| 洋葱块 | 30克| 辅料 |
| 青椒块 | 50克| 辅料 |
| 番茄块 | 100克| 辅料 |
| 蛋白 | 1个| 辅料 |
| 香芹 | 1棵| 辅料 |
| 黑胡椒酱 | 20克| 配料 |
| 橄榄油 | 20克| 配料 |
| 帕尔玛干酪粉 | 15克| 配料 |
| 意大利干香草碎 | 1克| 配料 |
| 柠檬汁 | 5克| 配料 |
