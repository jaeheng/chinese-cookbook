## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 无盐奶油 | 2大勺| 主料 |
| 色拉油 | 130ml| 主料 |
| 鸡蛋 | 3个| 主料 |
| 普通面粉 | 1杯| 主料 |
| 玉米淀粉 | 1杯| 主料 |
| 可可粉 | 0.25杯| 主料 |
| 泡打粉 | 0.5大勺| 主料 |
| 苏打粉 | 0.5小勺| 主料 |
| 红糖 | 180ml| 辅料 |
| 盐 | 0.5小勺| 辅料 |
