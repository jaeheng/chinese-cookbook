## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 双孢菇 | 6朵| 主料 |
| 金针菇 | 100克| 主料 |
| 培根 | 3片| 主料 |
| 黄瓜 | 60克| 主料 |
| 胡萝卜 | 60克| 主料 |
| 生菜 | 4片| 主料 |
| 姜 | 5克| 辅料 |
| 蒜 | 5克| 辅料 |
| 蚝油 | 一大勺| 配料 |
| 淀粉 | 一大勺| 配料 |
| 清水 | 60ML| 配料 |

## 步骤1

双孢菇6朵；金针菇100克；培根3片；黄瓜60克；胡萝卜60克；生菜4片；耗油一大勺；淀粉一大勺；清水60ML；姜、蒜各5克。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112029399.jpg?x-oss-process=style/p320) 

## 步骤2

黄瓜、胡萝卜切条；姜蒜切末；金针菇洗净切去根部。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112032076.jpg?x-oss-process=style/p320) 

## 步骤3

一片培根从中间切开，取一绺金针菇、一根黄瓜条、一根胡萝卜条各卷住。接口处用牙签固定，依次做好6个培根卷。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112037073.jpg?x-oss-process=style/p320) 

## 步骤4

入锅煎至两面金黄，金针菇变软，盛出备用。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112039377.jpg?x-oss-process=style/p320) 

## 步骤5

煎培根卷的油煸香姜蒜末。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112049211.jpg?x-oss-process=style/p320) 

## 步骤6

入双孢菇略翻炒，加入少许清水焖软。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112051596.jpg?x-oss-process=style/p320) 

## 步骤7

倒入耗油、淀粉、清水勾兑好的芡汁大火收浓。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112053324.jpg?x-oss-process=style/p320) 

## 步骤8

盛出摆盘，即可上桌。

![双菇鲜蔬培根卷](https://i8.meishichina.com/attachment/recipe/201010/201010112054280.jpg?x-oss-process=style/p320)
