## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 新鲜猪肚 | 一个| 主料 |
| 红辣椒 | 2个| 辅料 |
| 尖椒 | 2个| 辅料 |
| 豆瓣酱 | 适量| 配料 |
| 葱 | 适量| 配料 |
| 子姜 | 适量| 配料 |
| 大蒜 | 适量| 配料 |

| 醋 | 适量| 配料 |
| 鸡精 | 适量| 配料 |

| 蚝油 | 适量| 配料 |
| 花生油 | 适量| 配料 |

## 步骤1

要将猪肚上的污秽和臊味除去，把里面翻出来用盐腌一下加入，面粉把猪肚抓一遍，放在清水里洗去污秽粘液，然后放进开水锅中煮至白脐结皮取出，再放在冷水中泡一下，用刀刮去白脐上的秽物。用清水冲洗干净后放高压锅煮至熟烂，捞出切片。

## 步骤2

锅内油烧热放入豆瓣酱炒香，放葱姜蒜同炒，炒出香味后放入煮熟的猪肚翻炒，同时加入一点糖, 盐、酱油、蚝油,醋。

## 步骤3

放入红青椒翻炒，出锅前放入鸡精。可以出炉了~呵呵。
