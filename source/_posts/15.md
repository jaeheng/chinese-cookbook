## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 豆腐 | 200克| 主料 |
| 油菜 | 500克| 主料 |
| 猪里脊肉 | 适量| 主料 |
| 葱姜蒜 | 适量| 辅料 |
| 水淀粉 | 适量| 辅料 |
| 盐 | 适量| 配料 |
| 料酒 | 适量| 配料 |
| 酱油 | 适量| 配料 |
| 醋 | 适量| 配料 |
| 鸡粉 | 适量| 配料 |
| 麻油 | 适量| 配料 |

## 步骤1

豆腐切片，沾水淀粉煎至两面金黄。油温不用太高，小火就可以。

## 步骤2

油菜洗净切小段。里脊肉放适量料酒酱油腌制5分钟。葱姜蒜切碎备用。

## 步骤3

锅中入油，5分热时放入葱姜蒜炒香。

## 步骤4

将里脊肉下锅滑散，烹入几滴醋（可以保持肉的鲜嫩）。

## 步骤5

肉变色的时候放入刚才煎好的豆腐，翻炒几下倒入一小碗水。

## 步骤6

水开后放入切好的油菜（油菜梗在下面，油菜叶在上面），放入一勺盐，盖上锅盖焖5分钟。

## 步骤7

打开锅盖，这时菜已经基本完成，放入适量鸡粉炒匀，用水淀粉一层薄芡，临出锅时滴点麻油。完成！
