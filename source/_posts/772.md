## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 肉 | 一小块| 主料 |
| 地瓜 | 一个| 辅料 |
| 胡萝卜 | 一根| 辅料 |
| 木耳 | 两个| 辅料 |
| 油 | 两大勺| 配料 |
| 糖 | 一小勺| 配料 |
| 盐 | 一小勺半| 配料 |
| 鸡精 | 一小勺| 配料 |
| 料酒 | 一小勺| 配料 |
| 酱油 | 一大勺| 配料 |
| 生粉 | 适量| 配料 |
| 葱姜 | 适量| 配料 |

## 步骤1

将肉洗净后切成肉丝。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201416231.jpg?x-oss-process=style/p320) 

## 步骤2

把肉丝用一小勺料酒、一大勺酱油和适量的生粉拌匀后浆一会儿。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201417095.jpg?x-oss-process=style/p320) 

## 步骤3

地瓜去皮、洗净后切成丝。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201418039.jpg?x-oss-process=style/p320) 

## 步骤4

胡萝卜去皮、洗净后用刨子刨成丝。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201419289.jpg?x-oss-process=style/p320) 

## 步骤5

木耳洗净后切成丝。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201438349.jpg?x-oss-process=style/p320) 

## 步骤6

锅内放一大勺油，放入葱姜煸出香味。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201440078.jpg?x-oss-process=style/p320) 

## 步骤7

倒入肉丝，煸至变色后盛出。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201441179.jpg?x-oss-process=style/p320) 

## 步骤8

洗净锅，再放一大勺油。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201442049.jpg?x-oss-process=style/p320) 

## 步骤9

将地瓜丝、胡萝卜丝、木耳丝一起倒入翻炒片刻。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201442485.jpg?x-oss-process=style/p320) 

## 步骤10

倒入炒好的肉丝翻炒均匀。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201446098.jpg?x-oss-process=style/p320) 

## 步骤11

撒一小勺糖和鸡精、一小勺半盐调味，再翻炒片刻就可装盘了。

![丝丝小炒](https://i8.meishichina.com/attachment/recipe/201010/201010201450286.jpg?x-oss-process=style/p320)
