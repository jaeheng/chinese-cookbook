## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 杏鲍菇 | 3个| 主料 |
| 鱼肉 | 100克| 主料 |
| 五花肉 | 100克| 主料 |
| 马蹄 | 4个| 主料 |
| 香菇 | 1个| 主料 |
| 香菜 | 2棵| 辅料 |
| 上海青 | 3棵| 辅料 |
| 姜 | 25克| 辅料 |
| 胡椒粉 | 1/2茶匙| 辅料 |
| 鸡蛋 | 1个| 辅料 |
| 淀粉 | 1汤匙| 辅料 |
| 盐 | 2茶匙| 配料 |
| 鸡精 | 1/2茶匙| 配料 |
| 瑶柱鲍鱼汁 | 4汤匙| 配料 |

## 步骤1

主料图。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092253046.jpg?x-oss-process=style/p320) 

## 步骤2

五花肉、鱼肉剁肉馅。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010101016575.jpg?x-oss-process=style/p320) 

## 步骤3

香菇泡发后洗净切碎，马蹄、香菜剁碎，姜切粒待用。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092303118.jpg?x-oss-process=style/p320) 

## 步骤4

把香菇、马蹄、香菜、姜粒放入内馅中，加胡椒粉、盐、鸡精、鸡蛋清搅拌上劲。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092304402.jpg?x-oss-process=style/p320) 

## 步骤5

杏鲍菇切成大小相同的段，上海青切成四瓣。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092305259.jpg?x-oss-process=style/p320) 

## 步骤6

杏鲍菇从中间切开一刀，不切断。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092306108.jpg?x-oss-process=style/p320) 

## 步骤7

锅里放入清水烧开后，放1茶匙盐，杏鲍菇用开水先焯一下。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092307079.jpg?x-oss-process=style/p320) 

## 步骤8

捞起擦干水分。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092307361.jpg?x-oss-process=style/p320) 

## 步骤9

在菇的切口处抹上干生粉，再把肉馅酿在里面。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092308487.jpg?x-oss-process=style/p320) 

## 步骤10

放入锅里大火蒸10分钟取出。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092309306.jpg?x-oss-process=style/p320) 

## 步骤11

上海青去掉粗叶，用油盐水焯水。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092310503.jpg?x-oss-process=style/p320) 

## 步骤12

焯水后放入冰水中。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092321212.jpg?x-oss-process=style/p320) 

## 步骤13

杏鲍菇、上海青码好盘。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092321520.jpg?x-oss-process=style/p320) 

## 步骤14

瑶柱鲍鱼汁、水淀粉备好。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092323232.jpg?x-oss-process=style/p320) 

## 步骤15

锅下油倒入鲍鱼汁。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092324222.jpg?x-oss-process=style/p320) 

## 步骤16

加水淀粉煮滚。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092325394.jpg?x-oss-process=style/p320) 

## 步骤17

淋在杏鲍菇上即可。

![瑶柱鲍汁酿杏鲍菇](https://i8.meishichina.com/attachment/recipe/201010/201010092326551.jpg?x-oss-process=style/p320)
