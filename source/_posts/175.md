## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 大明虾 | 适量| 主料 |
| 蛋黄 | 适量| 辅料 |
| 奶酪片 | 适量| 辅料 |
| 牛奶 | 适量| 辅料 |
| 西兰花 | 适量| 辅料 |
| 香什草 | 适量| 辅料 |
| 盐 | 适量| 配料 |
| 糖 | 适量| 配料 |

## 步骤1

挑大的虾了入，剪掉虾脚和虾须，从背后下刀，取出沙筋洗净，撒一丁点盐备用。

## 步骤2

取2个蛋黄加入糖后边打边加入牛奶，慢慢的打成奶黄酱，直接由奶黄酱买就省电时间吧。

## 步骤3

将奶酪丝放入开背中，盖上一层厚厚的奶黄酱，放入烤盘中，200度10分钟左右。

## 步骤4

窜烫一下西兰花，用剩的奶黄酱可以配着西兰花吃，虾背上撒一下香什草，迷迭香之类的，黑胡椒碎也可，完成。
