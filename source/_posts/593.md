## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 干面条 | 适量| 主料 |
| 香菇 | 适量| 辅料 |
| 红萝卜 | 适量| 辅料 |
| 蚬子 | 适量| 辅料 |
| 香菜 | 适量| 配料 |
| 葱 | 适量| 配料 |
| 姜 | 适量| 配料 |
| 油 | 适量| 配料 |
| 盐 | 适量| 配料 |
| 胡椒粉 | 适量| 配料 |
| 鸡精 | 适量| 配料 |

## 步骤1

各式样材料洗净,香菇,红萝卜,姜切片,葱,香菜切段。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142156111.jpg?x-oss-process=style/p320) 

## 步骤2

锅里放水,将姜片,葱段放入烧开。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142156277.jpg?x-oss-process=style/p320) 

## 步骤3

水 开后将香菇,红萝卜,蚬子放入煮开。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142158086.jpg?x-oss-process=style/p320) 

## 步骤4

小火再煮5分钟,放盐,胡椒粉,鸡精,麻油调味。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142159109.jpg?x-oss-process=style/p320) 

## 步骤5

同时另一锅水烧开,将面条煮熟后捞入煮好的蚬子汤中,洒上香菜即可。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142159415.jpg?x-oss-process=style/p320) 

## 步骤6

出品。。。

![鲜味蚬子面](https://i8.meishichina.com/attachment/recipe/201010/201010142159557.jpg?x-oss-process=style/p320)
