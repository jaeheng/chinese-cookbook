## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 嫩豆腐 | 500克| 主料 |
| 肥肉 | 50克| 主料 |
| 鸡脯肉 | 50克| 主料 |
| 胡萝卜 | 30克| 辅料 |
| 大甜椒 | 20克| 辅料 |
| 香菌 | 15克| 辅料 |
| 豌豆苗 | 50克| 辅料 |
| 葱水 | 20克| 辅料 |
| 姜水 | 适量| 辅料 |
| 盐 | 3克| 配料 |

## 步骤1

豆腐揭去表皮后用箩筛制茸，用纱布搌干水分后置盆中。猪肥膘肉和鸡脯肉去筋膜捶茸，与豆腐同置一盆中，加葱姜水搅散后。再加盐和鸡蛋清制成掺。

![鲜花豆腐](https://i8.meishichina.com/attachment/recipe/201001/201001172058474.jpg?x-oss-process=style/p320) 

## 步骤2

在扇形与蝶形模具上抹一层猪化油，分别制出10个扇形、2个蝴蝶形的豆腐掺，将胡萝卜、大甜椒、香菌等片刻成不同花卉图案，分别嵌于豆腐掺上，上笼蒸熟备用。

![鲜花豆腐](https://i8.meishichina.com/attachment/recipe/201001/201001172059009.jpg?x-oss-process=style/p320) 

## 步骤3

炒锅置旺火上，加清汤烧沸，加胡椒粉，放入豌豆苗烫熟，舀入汤盆内，将蒸好的豆腐惨滑入汤内即成。
