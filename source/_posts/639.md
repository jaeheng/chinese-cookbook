## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 鸡腿 | 2支| 主料 |
| 番薯粉 | 250克| 主料 |
| 米酒 | 1大匙| 辅料 |
| 酱油 | 3大匙| 辅料 |
| 糖 | 1/2大匙| 辅料 |
| 葱 | 1支| 辅料 |
| 姜 | 2片| 辅料 |
| 椒盐 | 4大匙| 辅料 |

## 步骤1

鸡腿切成四小方块，用调味料（A）腌10分钟。

## 步骤2

腌好的鸡块沾上番薯粉少许，并用手捏紧。

## 步骤3

油6碗烧热后放下鸡块，炸至酥黄时捞出。

## 步骤4

趁热撒下椒盐，拌匀即可。
