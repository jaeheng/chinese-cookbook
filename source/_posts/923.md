## 准备材料

| 材料     | 用量 | 备注|
| ------- | ----- | --- |
| 牛肉 | 500克| 主料 |
| 姜 | 适量| 辅料 |
| 老抽 | 适量| 辅料 |
| 盐 | 适量| 辅料 |
| 五香粉 | 适量| 辅料 |
| 鸡精 | 适量| 辅料 |
| 糖 | 适量| 辅料 |
| 红酒 | 适量| 辅料 |

## 步骤1

用调料腌制好的牛肉（最好腌制5到7个小时）。

![用微波炉自制五香牛肉干](https://i8.meishichina.com/attachment/recipe/201010/201010251136098.JPG?x-oss-process=style/p320) 

## 步骤2

放在微波炉（用有盖子的容器盛牛肉）大火10分钟。

## 步骤3

将容器里德汤汁倒掉，小火10分钟。如果觉得软可以在 小火3分钟左右。根据自己的口感，千万别糊了啊。
